Enter search & replace query (or regex)

Search & Replace queries are case sensitive. The first input is what will be searched for (cloud) and the second input is what will be replaced (butt).

For Search & Replace queries, only whole words are matched. That is, if you want to replace cloud with butt, the string "Clouds are made of cloud groups" it will turn into "Clouds are made of butt groups". The plural clouds was not matched because it was not the whole word being searched for. To avoid this, use the regex filter.

Blank values won't be used. Downloads will appear below after generation.
